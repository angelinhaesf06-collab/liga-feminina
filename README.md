# LBT - Liga de Beach Tennis

Aplicativo mobile para organização e acompanhamento de torneios de Beach Tennis (Formato Rainha da Praia).

## 🚀 Objetivo
Foco em **agilidade e zero fricção** para organizadoras e jogadoras na beira da quadra.

## ✨ Premissas
- **Sem Login Tradicional:** Acesso via PIN de sala para jogadoras (Read-Only).
- **Controle Administrativo:** Local no dispositivo da organizadora.
- **Estética "Quiet Luxury":** Cores inspiradas na marca LBT (Marrom Café, Creme e Dourado).

## 🛠️ Stack Tecnológica
- **Frontend:** Expo (React Native) + TypeScript
- **Navegação:** Expo Router
- **Backend:** Supabase (PostgreSQL + Realtime)
- **Ícones:** Lucide React Native

## 📂 Estrutura do Projeto
- `src/app`: Rotas e telas do aplicativo.
- `src/constants`: Definições de cores e temas.
- `src/services`: Integração com serviços externos (Supabase).
- `assets`: Ícones e imagens (incluindo o ícone vetorial LBT).
